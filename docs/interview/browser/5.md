# 第 5 题：输入 URL 后回车发生什么

1. DNS 解析拿到 IP 地址
   1. 先从本地 DNS 服务器查找
   2. 找不到再从远程域名服务器查找
2. HTTP 请求
   1. 建立 TCP 连接：三次握手
   2. 发送 HTTP 请求，服务器处理请求，返回响应结果
   3. 关闭 TCP 连接：四次挥手
3. [浏览器渲染](./1.md)
   1. 构建 DOM 树
   2. 构建 CSSOM 树
   3. 将 DOM 树和 CSSOM 树结合，生成渲染树(Render Tree)
   4. 根据渲染树来绘制
