# 对象

从人类认知角度来讲，对象应该是以下事物之一：

  1. 一个可以触摸或者可以看见的东西；
  2. 人的智力可以理解的东西；
  3. 可以指导思考或行动（进行想象或施加动作）的东西。

# JavaScript 对象的特征

  • 对象具有唯一标识性：即使完全相同的两个对象，也并非同一个对象。
  • 对象有状态：对象具有状态，同一对象可能处于不同状态之下。
  • 对象具有行为：即对象的状态，可能因为它的行为产生变迁。

一般而言，各种语言的对象唯一标识性都是用内存地址来体现的， 对象具有唯一标识的内存地址，所以具有唯一的标识。
