# 第 3 题：COOKIE 和 SESSION 有什么区别

他们的出现主要是由于http是的无状态，服务端需要用户的状态，就需要通过某种机制来实现，这种机制就是Session。

  • Session： 在服务端保存的一种数据结构，用来跟踪用户状态。
  • Cookie： 在客户端保存的信息的一种机制，也是实现 session 的一种方式。
