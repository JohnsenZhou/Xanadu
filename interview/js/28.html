# 第 28 题： 排序算法题

# 题目

在一个字符串数组中有红、黄、蓝三种颜色的球，且个数不相等、顺序不一致，请为该数组排序。使得排序后数组中球的顺序为:黄、红、蓝。

例如：红蓝蓝黄红黄蓝红红黄红，排序后为：黄黄黄红红红红红蓝蓝蓝。

# 解答

const type = {
  红: 1,
  黄: 2,
  蓝: 3
};
function sort(str = "") {
  const arr = str.split("");
  return arr.sort((a, b) => type[a] - type[b]);
}
console.log(sort("红蓝蓝黄红黄蓝红红黄红"));
