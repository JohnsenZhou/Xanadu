# 第 17 题：如何把一个字符串的大小写取反

例如 ’AbC' 变成 'aBc'

# 解答

"HeheSb".replace(/[a-zA-Z]/g, word => {
  if (/[a-z]/.test(word)) {
    return word.toUpperCase();
  } else {
    return word.toLowerCase();
  }
});
