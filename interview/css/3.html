# 第 3 题：弹性盒子中 flex: 0 1 auto 表示什么意思

flex 属性是 flex-grow、flex-shrink、flex-basis 的简写。

  • flex-grow：项目放大比例，默认为 0，即如果存在剩余空间也不放大。
  • flex-shrink：项目缩小比例，默认为 1，即如果空间不足，将项目缩小。
  • flex-basis：在分配多余空间之前，项目占据的主轴空间。
